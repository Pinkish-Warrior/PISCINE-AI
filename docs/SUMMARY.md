## 📋 Executive Overview

**AI.GO** is a comprehensive JavaScript-based web development curriculum offered by 01-edu, designed to teach fundamental programming concepts through interactive, project-based learning. The course uses a gamified approach where students build and animate a virtual robot while learning HTML, CSS, and JavaScript.

---

## 🎯 Course Structure & Roadmap

### Learning Path (Sequential Progression)

The AI.GO curriculum consists of **25+ progressive exercises** organized in a logical learning sequence:

#### **Phase 1: Foundations (HTML & CSS)**
1. **first-hello** - Create a button that displays/hides "Hello World" text
2. **first-move** - Animate robot movement using CSS transforms
3. **first-wink** - Add interactive animations (winking, blinking)
4. **first-function** - Introduction to JavaScript functions

#### **Phase 2: Core JavaScript Concepts**
5. **declare-everything** - Variable declaration and scope
6. **play-with-variables** - Variable manipulation and data types
7. **i-win-arguments** - Function parameters and arguments
8. **good-recipe** - Function best practices and code organization
9. **only-if** - Conditional statements (if/else)
10. **listed** - Working with arrays and lists

#### **Phase 3: DOM Manipulation & Events**
11. **objects-around** - JavaScript objects and properties
12. **call-it** - Function invocation and callbacks
13. **class-it** - CSS classes and dynamic styling
14. **select-then-style** - DOM selection and CSS manipulation
15. **colorful-arms** - CSS animations and transformations
16. **colorful-legs** - Advanced styling techniques

#### **Phase 4: Advanced Concepts**
17. **embedded-organs** - Nested elements and complex DOM structures
18. **glance-on-power** - Operators and expressions
19. **robots-harmony** - Multiple robots and state management
20. **transform-objects** - Object transformations and methods
21. **star-forge** - Complex animations and effects
22. **the-skeleton** - HTML semantic structure
23. **the-smooth-operator** - Animation smoothing and timing
24. **JSPowered Mode** - Advanced JavaScript concepts
