# PiscineAI — AI.GO

Beginner-friendly path through the [01-edu AI.GO curriculum](https://github.com/01-edu/public/tree/master/subjects/AI.GO).

## Start here

1. **[docs/SETUP.md](docs/SETUP.md)** — one-time technical setup: install VS Code, install Git, sign up for an AI assistant, clone the repo, run a static site from the terminal.
2. **[docs/ROADMAP.md](docs/ROADMAP.md)** — the full learning path as a Mermaid diagram (setup → 4 phases → 23 exercises → done).
3. **[docs/ROADMAP_BEGINNERS.md](docs/ROADMAP_BEGINNERS.md)** — the same roadmap as tables, with per-phase checkpoints, suggested pacing, and study tips.
4. **[docs/SUMMARY.md](docs/SUMMARY.md)** — the original repo analysis this roadmap was built from (course structure, tech stack, prerequisites).

For the exercise tasks themselves (goal, HTML/CSS/JS to add, expected result), open the matching folder in [`01-edu/public/subjects/AI.GO`](https://github.com/01-edu/public/tree/master/subjects/AI.GO) — that's the source of truth, not duplicated here.
