# PiscineAI — AI.GO

Beginner-friendly path through the [01-edu AI.GO curriculum](https://github.com/01-edu/public/tree/master/subjects/AI.GO). No prior coding experience needed.

## Start here

1. **[docs/SETUP.md](docs/SETUP.md)** — do this first. One-time setup: install VS Code, install Git, sign up for an AI assistant, clone the repo, run a static site from the terminal.
2. **[docs/WORKFLOW.md](docs/WORKFLOW.md)** — your day-to-day guide. All 24 exercises as tables, grouped into 4 phases, with checkpoints, suggested pacing, and study tips.
3. **[docs/ROADMAP.md](docs/ROADMAP.md)** — the same path as a Mermaid diagram, if you'd rather see the whole flow at a glance.

For the exercise tasks themselves (goal, HTML/CSS/JS to add, expected result), open the matching folder in [`01-edu/public/subjects/AI.GO`](https://github.com/01-edu/public/tree/master/subjects/AI.GO) — that's the source of truth, not duplicated here.

It is live here: [Piscine-AI](https://pinkish-warrior.github.io/PISCINE-AI/)
